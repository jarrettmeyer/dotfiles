# dotfiles

These are my dotfiles.

## Prerequisites

+ `git`.
+ `zsh`. This will probably work with `bash`, too, but I make no promises.

## Usage

This one is easy.

```
$ git clone git@github.com:jarrettmeyer/dotfiles.git
$ cd dotfiles
$ ./setup.sh
```
